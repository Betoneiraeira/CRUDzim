# CRUD de Alunos - Projeto de Estudo

## ❓ O que faz?
O aluno cadastra suas informações como nome, email e idade. Essas informações vão para o banco de dados onde será atribuído uma matrícula de forma automática a partir do número 1000. Também é possível editar as informações e excluir os usuários.

## 🛠️ Tecnologias usadas
- PHP
- MySQL
- HTML/CSS

## 🚀 Como rodar o projeto
1. Importe o arquivo `escola.sql` no seu MySQL (phpMyAdmin ou linha de comando)
2. Configure a conexão no arquivo `conexao.php`
3. Execute o projeto em um servidor local (XAMPP/WAMP)

## ⚠️ Atenção
**Todas as informações são fictícias**, este projeto foi feito apenas para fins de estudos e pode sofrer mudanças extremas futuramente.
